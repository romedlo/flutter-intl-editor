# Flutter Intl Editor

A simple and intuitive VS Code extension for managing Flutter internationalization (.arb) files in a visual way.

## Features
- View and edit .arb files in a table-like interface
- Easily manage translation keys and values
- Supports multiple languages
- Simplifies localization workflow for Flutter projects

## Usage
1. Open your Flutter project
2. Locate your .arb files (e.g., app_en.arb, app_es.arb)
3. Open any .arb file
4. Launch the editor interface from the extension

## Configuration

The extension automatically detects .arb files based on their filename (e.g., `_en`, `_es`).

## Requirements
- A Flutter project using internationalization (.arb files)

## Notes
- Make sure your .arb files follow the standard Flutter naming convention
- Changes are reflected directly in the files
